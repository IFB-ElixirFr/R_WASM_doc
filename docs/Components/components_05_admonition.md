# Notes

## Goal

Colored section according to type to inform users

## Code

```
::Admonition
---
type: note
text: "Le contenu de a_plus_b n'est pas modifié"
---
::
```

Type of admonition :

- note
- abstract
- info
- tip
- success
- question
- warning
- failure
- danger
- bug
- example
- quote

## Result

!!! note
    Le contenu de a_plus_b n'est pas modifié
